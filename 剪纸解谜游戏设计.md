**User**

一把剪刀一张红纸，折三折剪两刀，展开居然是朵牡丹花——这个从折叠到展开的"魔法"过程，能不能做成让人上瘾的解谜游戏？玩家面对一块虚拟红纸，需要选择折叠方式（对折、三角折、四折、六角折、窗花折、团花折6种基本折法），然后在折好的纸上划定剪裁区域，点击"展开"看结果是否匹配目标图案。共设计60关，从简单的单折单剪到复杂的多折组合剪裁，每关给出目标展开图和可用折法提示。评分看三方面：步骤数越少分越高、纸张利用率（保留面积/总面积）越高越好、用时越短分越高，三档分别评星。每10关引入一种新折法，难度递进要平滑不能出现跳跃感。展开算法是核心：用户选好折法和剪裁区域后，系统用数学矩阵变换模拟纸张折叠，再用镜像对称迭代计算展开后的完整图案，这个过程要实时可视化——先播折纸动画再播展开动画，让玩家直观理解原理。美术用CSS和SVG绘制剪纸风格UI，红色宣纸质感背景，金色线条装饰。技术选Phaser 3做游戏框架，折叠逻辑用矩阵变换，展开算法用镜像对称迭代渲染到Canvas。交付物有完整游戏、60关数据配置文件、折叠展开算法核心模块、关卡选择与星级评价页、排行榜，以及docker-compose配置。

### 核心模块说明

**1. 折叠展开算法** ([algorithm.js](file:///d:/bz/605/6057/js/algorithm.js))
- 7种折法：对折(纵/横)、三角折、四折、六角折、窗花折、团花折
- **矩阵变换折叠**：每种折法定义折叠区域的坐标映射
- **镜像对称迭代展开**：通过旋转对称(Cn)和二面体对称(Dn)群运算，将折叠纸上的剪裁区域映射回完整图案
- 80×80网格精度，支持矩形/圆形/三角形三种剪裁形状
- 实时渲染到Canvas，支持折叠视图和展开视图

**2. 60关关卡数据** ([levels.js](file:///d:/bz/605/6057/js/levels.js))
- 第1-10关：**对折篇**（双边对称入门）
- 第11-20关：**三角折篇**（对角镜像）
- 第21-30关：**四折篇**（四重对称）
- 第31-40关：**六角折篇**（六重旋转，雪花图案）
- 第41-50关：**窗花折篇**（八重二面体对称，传统窗花）
- 第51-60关：**团花折篇**（十二重旋转，牡丹花等复杂图案）
- 每10关引入新折法，难度平滑递进

**3. 游戏场景** ([game.js](file:///d:/bz/605/6057/js/game.js))
- **BootScene → MenuScene → LevelSelectScene → GameScene → ResultScene + LeaderboardScene**
- 主菜单：剪纸风格标题 + 折纸加载动画
- 关卡选择：60关网格 + 篇章分组 + 星级显示 + 解锁机制
- 游戏界面：目标图案(左) + 操作纸(中) + 折法/工具/操作(右)
- 折纸动画 + 展开动画（Phaser Tween）
- 矩形/圆形/自由三种剪裁工具
- 撤销/重置功能

**4. 评分系统**
- 三维度评星（各1-3星，共9星）：
  - **步骤数**：越少星越高
  - **纸张利用率**：保留面积/总面积，越高星越高
  - **用时**：越短星越高
- 匹配度≥85%即通关

**5. 排行榜**
- localStorage持久化存储
- 按星级、步骤、用时综合排序
